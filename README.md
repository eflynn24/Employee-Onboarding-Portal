**Overview**

The Employee Onboarding Portal is a responsive web-based application designed to streamline the employee onboarding process. It provides an organized, tab-based interface for collecting employee information, managing onboarding tasks, and guiding new hires through required documentation and setup activities.

The application is built using standard HTML and CSS, making it easy to customize for different organizational needs.

**Features**

Tabbed Navigation

Modern User Interface

Organized onboarding workflow

Active tab highlighting

Personal information collection

Employment-related information

Mobile-friendly layout

Multi-field form layouts

**Planned Backend Features**

Employee record management (SQL)

Employee Information Forms

Database-backed form submissions

Employee onboarding status tracking

Link required tasks/steps in tabs

Document management and file upload

Frontend	HTML, CSS, JavaScript

Backend API	Python

PostgreSQL Database
Hosting	Azure
Authentication	JWT
